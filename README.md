# Tennessee Public School Data Tools

Python notebooks for analyzing Tennessee K-12 education data, including state assessment scores, school letter grades, and enrollment projections.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `3-8_QuickScores.ipynb` | Quick analysis of grades 3-8 TCAP assessment scores |
| `EOC_Quickscores.ipynb` | End-of-Course exam score analysis |
| `EOC Files by Teacher.ipynb` | EOC results organized by teacher for evaluation support |
| `Calculate_AMO.ipynb` | Calculate Annual Measurable Objectives (AMO) targets |
| `Create all Proficiencies.ipynb` | Generate proficiency reports across all tested subjects |
| `Letter_Grade_Analysis.ipynb` | Analysis of Tennessee school letter grade components |
| `4th_Grade_Projection_File_Organization.ipynb` | Organize 4th grade projection data for middle school planning |

## Requirements

- Python 3.8+
- pandas
- openpyxl
- jupyter

## Context

These notebooks work with data from the [Tennessee Department of Education](https://www.tn.gov/education/districts/data/data-downloads.html) and PowerSchool SIS exports. They're designed for district-level data analysts and school administrators working with TN-specific assessment frameworks (TCAP, EOC, Ready Graduate).
